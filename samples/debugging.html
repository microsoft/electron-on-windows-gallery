Debugging

Instructions for debugging Electron apps on Windows.

Debugging Tools

Use these tools to debug your Electron applications:

  • Chrome DevTools
  • VS Code Debugger
  • Visual Studio Debugger
  • Windows Performance Analyzer

Main Process Debugging

Debug the main process using VS Code or the built-in Node.js debugger.

Renderer Process Debugging

Use Chrome DevTools to debug renderer processes in your Electron application.
